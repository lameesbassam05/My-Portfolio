# My Portfolio Website

This is a simple personal portfolio website created using HTML only.

## 📌 Project Description
The website contains two pages:
- Home page (index.html)
- Contact page (contact.html)

It presents my personal information, skills, projects, and social media accounts.

## 🧑‍💻 Features
- Personal information section
- Social media links (open in new tab)
- Skills (technical & life skills)
- Projects table with images
- Contact form (name, email, gender, message)

## 🛠️ Technologies Used
- HTML
